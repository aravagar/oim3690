## What I learned

### CSS properties and concepts

- `box-sizing: border-box` makes width and height include padding and border, which makes layouts far easier to reason about.
- Flexbox is for one-dimensional layouts. `display: flex` plus `justify-content` and `align-items` handled my navbar.
- Grid is for two-dimensional layouts. `grid-template-columns: repeat(auto-fit, minmax(240px, 1fr))` gives a product row that reflows on its own without me writing breakpoints for every screen size.
- `aspect-ratio` and `object-fit: cover` keep product images the same shape without squashing them.
- `position: fixed` for a navbar that stays in place, and CSS variables in `:root` so I only define a color once and reuse it with `var()`.
